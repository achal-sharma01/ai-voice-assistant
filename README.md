# 🤖 Astra – AI Voice Assistant for Smart Room Automation

An AI-powered smart room automation system featuring natural voice interaction, real-time monitoring, cloud-based synchronization, and automated reporting through a centralized web application.

---

## 🧾 Overview

Astra is an AI-powered voice assistant developed to automate smart room operations using natural voice commands. The system integrates voice recognition, cloud synchronization, environmental monitoring, and automated reporting to provide a seamless and interactive automation experience.

---

## 🛠 Technologies Used

- Python
- Firebase Realtime Database
- Web Speech API
- HTML, CSS & JavaScript
- Google Apps Script

---

## ✨ Key Features

- AI-powered voice commands for smart device control
- Real-time synchronization using Firebase Realtime Database
- Live monitoring of room conditions and device status
- Environmental monitoring (temperature, humidity, and occupancy)
- Event logging and activity history
- AI-generated daily operational reports
- Manual control panel with voice and button interactions

---

## 🔄 System Workflow

1. Capture user voice commands using speech recognition.
2. Process commands through the AI voice assistant.
3. Synchronize live device status and sensor data with Firebase.
4. Display real-time operational data through the web application.
5. Generate automated daily reports based on recorded events and system activity.

---

## 📈 Project Highlights

- Developed an AI-powered web application for voice-controlled smart room automation.
- Integrated real-time cloud synchronization with Firebase Realtime Database.
