# 🤖 Astra – AI Voice Assistant for Smart Room Automation

An AI-powered smart room automation system featuring natural voice interaction, real-time monitoring, cloud-based synchronization, and automated reporting through a centralized web application.

---

## 🧾 Overview

Astra is an AI-powered voice assistant developed to automate smart room operations using natural voice commands. The system combines speech recognition, cloud synchronization, environmental monitoring, and automated reporting to provide an interactive and intelligent automation experience.

The application enables users to control room devices through voice commands while monitoring real-time environmental data such as temperature, humidity, occupancy, and device status. It also maintains operational history and generates AI-powered daily reports for monitoring and analysis.

---

## 🚀 Key Features

- 🎙 AI-powered voice commands for smart device control
- ☁ Real-time synchronization using Firebase Realtime Database
- 📊 Live monitoring of room conditions and device status
- 🌡 Environmental monitoring (Temperature & Humidity)
- 👥 Occupancy tracking
- 📝 Event logging and operational history
- 🤖 AI-generated daily operational reports
- 🎛 Manual control panel with voice and button interactions

## 🏗 Project Evolution

This project was developed in two phases.

### Phase 1
A Python-based AI voice assistant prototype was developed to experiment with speech recognition, text-to-speech, and automation logic.

### Phase 2
The system was redesigned as a cloud-connected web application using Firebase and Web Speech APIs, enabling real-time synchronization, improved accessibility, and a better user experience for the final implementation.

---

## 🔄 System Workflow

1. Capture user voice commands through speech recognition.
2. Process commands using the AI voice assistant.
3. Synchronize device status and environmental data with Firebase Realtime Database.
4. Display live operational information within the web application.
5. Log events and generate AI-powered daily operational reports.
