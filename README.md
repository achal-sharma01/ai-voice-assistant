🤖 Astra – AI Voice Assistant for Smart Room Automation

An AI-powered smart room automation system featuring natural voice interaction, real-time monitoring, cloud-based synchronization, and automated reporting through a centralized web application.

🧾 Overview

Astra is an AI-powered voice assistant developed as a final-year engineering project to automate room operations using natural voice commands. The system combines voice recognition, cloud synchronization, live monitoring, and automated reporting to provide a smart and interactive automation experience.

🛠 Technologies Used
Python
Firebase Realtime Database
Web Speech API
HTML, CSS & JavaScript
Google Apps Script
✨ Key Features
AI-powered voice commands for smart device control
Real-time synchronization using Firebase Realtime Database
Live monitoring of room conditions and device status
Environmental monitoring (temperature, humidity, occupancy)
Event logging and activity history
AI-generated daily operational reports
Manual control panel with voice and button interactions
🔄 System Workflow
Capture voice commands using speech recognition.
Process commands through the AI assistant.
Synchronize device status and sensor data with Firebase.
Display live operational information in the web application.
Generate automated daily reports based on recorded events.
📈 Highlights
Developed a centralized AI-driven web application for smart room automation.
Integrated voice recognition, cloud data synchronization, and automated reporting into a single platform.
Enabled real-time operational monitoring through live device and environmental data.
Designed to improve automation, monitoring, and reporting efficiency.
